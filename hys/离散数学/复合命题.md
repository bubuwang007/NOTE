令$p、q$为命题。

非$p\iff\neg p$

|  p   | $\neg p$ |
| :--: | :------: |
|  T   |    F     |
|  F   |    T     |



合取$\iff p\land q\iff$且

|  p   |  q   | $p\land q$ |
| :--: | :--: | :--------: |
|  T   |  T   |     T      |
|  T   |  F   |     F      |
|  F   |  T   |     F      |
|  F   |  F   |     F      |



析取 $\iff p\vee q\iff$或

|  p   |  q   | $p\vee q$ |
| :--: | :--: | :-------: |
|  T   |  T   |     T     |
|  T   |  F   |     T     |
|  F   |  T   |     T     |
|  F   |  F   |     F     |



异或 $\iff p\oplus q$

|  p   |  q   | $p\oplus q$ |
| :--: | :--: | :---------: |
|  T   |  T   |      F      |
|  T   |  F   |      T      |
|  F   |  T   |      T      |
|  F   |  F   |      F      |



条件语句 $\iff p\rightarrow q$

|  p   |  q   | $p\rightarrow q$ |
| :--: | :--: | :--------------: |
|  T   |  T   |        T         |
|  T   |  F   |        F         |
|  F   |  T   |        T         |
|  F   |  F   |        T         |



双条件$\iff p\leftrightarrow q$ 

|  p   |  q   | $p\leftrightarrow q$ |
| :--: | :--: | :------------------: |
|  T   |  T   |          T           |
|  T   |  F   |          F           |
|  F   |  T   |          F           |
|  F   |  F   |          T           |



优先级

|      运算符       | 优先级 |
| :---------------: | :----: |
|      $\neg$       |   1    |
|      $\land$      |   2    |
|      $\vee$       |   3    |
|   $\rightarrow$   |   4    |
| $\leftrightarrow$ |   5    |
